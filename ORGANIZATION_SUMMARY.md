  - Troubleshooting
  - Technical documentation
- **Business Docs** → `docs/business/`
  - Project summaries
  - ROI analysis
  - Portfolio materials

### **2. Script Organization**
- **Main Pipeline** → `scripts/run_pipeline.py`
- **Quick Start Scripts** → Root directory
  - `quick_start.sh` (Unix/Mac)
